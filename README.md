## ⚙️ Local Setup

Clone the repository:

```bash
git clone https://github.com/shivam-js/Me-Shivam-Portfolio-V2.git
```

Move into the project:

```bash
cd Me-Shivam-Portfolio-V2
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
